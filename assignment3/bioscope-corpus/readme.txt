BioScope corpus converted into conll format.

Column 1: doc id
Column 2: sentence id
Column 3: token id
Column 4: word
Column 5: negation cue label (B-negation, I-negation)
Column 6: scope label (B-NEG, I-NEG)

Publication BioScope corpus:

Veronika Vincze, György Szarvas, Richárd Farkas, György Móra & János Csirik. The BioScope corpus: biomedical texts annotated for uncertainty, negation and their scopes.   BMC Bioinformatics volume 9, Article number: S9 (2008)
